# DSSkillV2 Install

## Files

- Skill folder: `dsskillv2/`
- Zip package: `dsskillv2-skill-package.zip`

## Install

1. Unzip `dsskillv2-skill-package.zip`.
2. Copy the `dsskillv2` folder into your local skills directory.
3. Restart the agent session or reload skills.

## Use

- Invoke with `$dsskillv2`
- Best for projects that use one canonical token source and sync tokens to Token Studio, Figma, runtime, or AI consumers.

## Notes

- This package has no hardcoded dependency on James Hou's local filesystem.
- It works best in repositories that already have a token source plus adapter outputs.
